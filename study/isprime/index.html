소수 판별하기

에라토스체네스의 체

  • 어떤 수(N)까지의 소수 리스트 구하기
  1. 2부터 찾고자 하는 수까지를 모두 나열한다.
  2. 2부터 시작해서, 자신은 포함하되 자기 자신의 배수는 모두 제외한다.
  3. 찾고자 하는 수의 제곱근까지 2를 반복한다.

예시

  1. N = 16 : 16까지의 소수를 구하고 싶다
  2. 2부터 16까지 수를 나열한다.
  3. 2는 남기고, 2의 배수를 제외한다
  4. 3도 남기고, 3의 배수를 제외한다
  5. 4는 이미 없는 수(2에서 제거됨)이며, N의 제곱근이므로 종료한다.
  6. 남은 수는 [2, 3, 5, 7, 11, 13]

1. 어떤 수 n이 소수인가? 체크하기

import math

def IsPrime(n):
    if n == 1:
        return False
    
    for i in range(2, int(math.sqrt(n)) + 1):
        if n % i == 0:
            return False
    
    return True

2. 정수 n까지의 소수 리스트 구하기

def prime_list(n):
	lst = [True] * (n+1)  # 전부 소수라고 깔고 시작
	
	m = int(n ** 0.5) # 배수 판정 설정

	# 2부터 탐색
	for i in range(2, m+1):
		# 배수를 제외함
		if lst[i] == True:
			for j in range(i+i, n, i):
				lst[j] = False

	return [i for i in range(2, n+1) if lst[i] == True]
