[Hugo] 공통 프론트매터 만들기

hugo new 명령어로 새 글 작성 시 뜨는 프론트매터 수정하기

2023-04-19
Hugo
  • 어떤 글을 작성할 때, 글과 글 목록의 인터페이스가 다른 경우가 있었다. 예를 들면 글 목록에서는 오른쪽 사이드바가 뜨는데, 글에 들어가면 오른쪽 사이드바가 뜨지 않는 경우가 있었다.
  • mainroad 테마의 경우, 글 페이지에서의 사이드바는 프론트매터에서 관리되고 있다.

1. hugo new <폴더이름>/<문서이름.md>를 쉘에서 입력하고 파일을 연다.

---
title: "Test3"
date: "2023-04-19T16:48:04+09:00"
draft: true
---
  • 글의 상단에 ---로 감싸져 있는 부분이 프론트매터이며, 여기서 페이지를 조절할 수 있다.

2. mainroad 테마는 공식 레포지토리에 프론트매터에 들어갈 패러미터 예시가 있다.

  • Common-Defined Params만으로 참고가 될 것이다.
  • 다른 테마의 구조를 변경하고 싶다면 테마 문서에서 찾아보자.

3. config.toml 수정하기

  • 기본적으로 프론트매터는 <프로젝트명>/archetypes/default.md 혹은 <프로젝트명>/themes/<테마명>/archetypes.default.md에서 관리되는 것 같다.
  • 내 경우 전자의 default.md는 없었고 후자에는 있었는데, 이를 수정했다.
  • 수정을 끝냈다면, config.toml 파일에서 themes/<테마이름>/archetypes 경로를 지정해준다.
# config.toml
...
archetypeDir = "themes/mainroad/archetypes"
...

4. 결과 확인하기

  • hugo new <폴더이름>/<문서이름.md>를 입력해서 지정한 프론트매터가 잘 뜨는지 확인한다.
