[Python] 재귀

2023-04-21
코테

1. 재귀 함수란?

자기 자신을 호출하는 함수를 말한다.

def dfs(node):
  visited[start_node] = 1
  for i in graph[start_node]:
    if visited[i] == 0:
      dfs(i) # 함수 자신을 호출

일단은 dfs 내에서 dfs가 또 호출되는 것에만 주목하면 되겠다.

중요한 건 2가지다.

  1. 문제 정의 표기
  • 이건 모든 문제에서 중요하고 어렵다.
  1. 재귀 종료 조건
  • 위의 DFS처럼 진행 상황에 맞춰 조건을 추려나갈 수도 있고,
  • 아래의 예제처럼 함수가 최종적으로 도달하는 곳에서 반환값을 줄 수도 있다.

관련 용어

관련 링크
함수가 계속 함수를 호출하는 과정에서 각 함수가 저장되는 공간을 스택 프레임이라고 한다.
종결 조건이 있다면 가장 마지막에 호출된 함수부터 메모리를 반환한다.
만약 재귀 호출이 스택 프레임이 처리할 수 있는 용량을 벗어나면 이를 스택 오버플로우라고 한다.

2. 예시

  • 백준 피보나치 수 5 문제

$F_{n} = F_{n-1} + F_{n-2}$

  1. n번째 피보나치 수는 위 식처럼 나타날 것이다.
  2. n-1번째 피보나치 수와 n-2번째 피보나치 수도 위 식처럼 나타낼 수 있다
  3. 이런 식으로 $F_n$을 계속 쪼개나가면 $F_0$과 $F_1$의 합들로 이뤄지게 된다.

코드

def func(n):
  if n == 0:
    return 0
  elif n == 1:
    return 1
  
  return func(n-1) + func(n-2)

3. 관련 문제

백준 단계별 풀이 : 재귀

  • 이외에도 재귀는 많은 문제에서 활용된다.
  • 백준의 경우 재귀 호출의 상한을 별도로 줘야 하는 문제들이 있다. 아래 코드로 재귀 횟수의 상한을 변경할 수 있다.
import sys
sys.setrecursionlimit(int(1e6))
